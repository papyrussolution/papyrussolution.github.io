# Введение в Svelte

Svelte — это революционный подход к созданию веб-приложений...

## Основные преимущества

- **Компиляция** вместо рантайма
- **Меньше кода** чем в других фреймворках
- **Высокая производительность**

## Пример кода

```svelte
<script>
  let count = 0;
</script>

<button on:click={() => count++}>
  Кликов: {count}
</button>
